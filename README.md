Enfys is a WebGPU 3D graphics engine written in TypeScript.

## Running

Start the builder

```
npm run watch
```

Start the dev webserver

```
npm run start
```

Open http://localhost:3333

## Documentation

Build the documentation using

```
npm run docs
```

Open http://localhost:3333/docs/

## Tests

Run the test suite

```
npm test
```
